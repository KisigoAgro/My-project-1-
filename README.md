# 🎮  3D Prototype Shooter with Survival Elements

Это рабочий вертикальный срез (MVP) 3D-шутера от первого лица, созданный на движке **Unity 6** с использованием **Universal Render Pipeline (URP)**. Проект демонстрирует базовые навыки архитектуры кода, работы с искусственным интеллектом, системами ввода и контроля данных между игровыми сценами.

## 🛠️ Реализованные механики и технологии
* **Система ввода:** Полный переход на новую **Unity Input System**. Реализована блокировка управления и камеры при наступлении игровых событий.
* **Боевая система (Raycast):** Стрельба через физические лучи с расчетом дистанции и мгновенным нанесением урона врагам.
* **Искусственный Интеллект (AI Navigation):** Враги используют современную систему навигации Unity 6 (`NavMesh Surface`, `NavMesh Obstacle` с запеканием динамических зон `Carve`). Реализовано преследование и логика атаки игрока с кулдауном.
* **Архитектура и статические данные (C#):** Реализован бесшовный перенос данных инвентаря (`List<string>`) и здоровья между сценами с использованием статической памяти. Безопасное изменение здоровья защищено через свойства (`get; private set;`).
* **Игровой цикл (Core Loop):** Боевая локация (сбор лута с поверженных врагов) ➡️ Зона эвакуации ➡️ Сцена Убежища, где через триггер верстака списываются ресурсы и навсегда увеличивается максимальный запас HP игрока.
* **Интерфейс (UI):** Динамический интерфейс шлема, экраны победы/поражения и управление игровым временем (`Time.timeScale`) в рамках одного Canvas (оптимизация Draw Calls).

## 🚀 Как запустить проект
1. Склонируйте репозиторий.
2. Откройте проект через Unity Hub (Рекомендуемая версия: Unity 6+).
3. Добавьте сцены `BattleScene` и `Hideout` в **Build Settings**.
4. Запустите игру со сцены `BattleScene`.
